Estamos desplegando en minutos

Este es un ejemplo de sitio estático listo para subir a GitHub Pages o Netlify. Incluye Tailwind por CDN y un poco de JavaScript.

Ver características

Sin build

Usamos Tailwind por CDN (Red de Entrega de Contenido), así que no necesitas Node ni compilar.

Ligero

Entorno de pruebas: Solo HTML, CSS y JS. Ideal para demos, portafolios o documentación.

Fácil de desplegar

Funciona perfecto con GitHub Pages o Netlify Drop (drag & drop).

Contacto

Este formulario es de ejemplo (no envía datos a un backend).
